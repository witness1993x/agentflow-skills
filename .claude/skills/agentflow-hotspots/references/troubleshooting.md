# `agentflow-hotspots` troubleshooting

## "0 hotspots returned"

First-run case: `af hotspots` creates the daily file even when empty. Check `~/.agentflow/sources.yaml` — empty `twitter_handles` + `rss_feeds` means there's nothing to scan. Pass `--from-file assets/sources.yaml` after filling it in.

## "filter matched 0 / N"

Filter regex too narrow. Widen it. The skill should already warn the user, but if not: try shorter regex or drop the filter and use the unfiltered list.

## "missing publisher_account.brand / voice / output_language / product_facts / keyword_groups.core / search_queries"

Profile is incomplete and won't yield production-quality scans. Two paths:

```bash
af topic-profile init -i --profile <id>                           # interactive
af topic-profile init --profile <id> --from-file ../agentflow/assets/topic_profile.yaml   # from yaml
af topic-profile derive --profile <id>                             # LLM-derive from default_description
```

Then re-scan.

## "freshness scores all 0"

D1's freshness scorer needs at least one `source_references` entry per cluster. If your sources are empty / unreachable, all scores collapse to 0. Verify Twitter handles still exist; verify RSS feeds return XML.

## "Twitter scrape returning 401"

Twitter's free API tier throttles aggressively. Check `TWITTER_BEARER_TOKEN` in `.env`. The scraper falls back to RSS-style mirrors when the bearer token is missing or rate-limited.

## "hotspots file present but `af hotspot-show <id>` says not found"

Two storage paths exist (`hotspots/` and `search_results/`). The CLI checks both, but if the daily file was generated by an older runtime version it may be missing the index entries `hotspot-show` expects. Try `af hotspots --force-rescan` to regenerate.

## "constraint suggestion blocked Gate A"

`af hotspots` may surface `Config Suggestions` cards (e.g. "your profile is missing tag X commonly seen in similar publishers"). Review via:

```bash
af topic-profile suggestion-list --profile <id>
af topic-profile review <suggestion_id>
af topic-profile apply <suggestion_id>     # or reject
```

## Generic non-zero exit

```bash
tail -n 20 ~/.agentflow/logs/agentflow.log
```
